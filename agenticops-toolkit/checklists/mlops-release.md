# MLOps Release Checklist

## Model

- [ ] Training data version is recorded.
- [ ] Feature definitions are versioned.
- [ ] Metrics are logged.
- [ ] Baseline comparison is available.
- [ ] Model artifact is registered or stored.

## Serving

- [ ] Prediction schema is documented.
- [ ] Input validation is implemented.
- [ ] Error responses are predictable.
- [ ] Latency budget is measured.
- [ ] Rollback path is documented.

## Operations

- [ ] CI/CD pipeline passes.
- [ ] Container image builds.
- [ ] Deployment manifests are reviewed.
- [ ] Monitoring metrics are defined.
- [ ] Drift or quality checks are planned.
