# Roadmap

## Goal

Build a portfolio in 7-9 weeks that demonstrates the skills required for a Junior Machine Learning Engineer role in an AI Delivery Center.

## Week 1: AgenticOps Toolkit

Deliverables:

- monorepo structure;
- Python project configuration;
- GitHub Actions CI;
- AI-assisted coding prompt library;
- engineering checklists;
- reusable project templates;
- documented process for using Codex, Copilot, Claude Code, or similar tools.

## Weeks 2-3: Credit Risk and Fraud ML Platform

Deliverables:

- synthetic or public dataset;
- pandas and Spark pipeline;
- scikit-learn baseline model;
- MLflow experiment tracking;
- FastAPI model serving;
- PostgreSQL audit and prediction tables;
- Docker Compose setup;
- Kubernetes/OpenShift deployment manifests.

## Weeks 4-5: Banking RAG Agent

Deliverables:

- synthetic banking documents;
- retrieval pipeline;
- LangGraph workflow;
- tool calls to PostgreSQL and MongoDB;
- response citations;
- guardrails;
- evaluation set and automated quality checks.

## Weeks 6-7: Customer Intelligence Streaming

Deliverables:

- Kafka event ingestion;
- Spark Structured Streaming transformations;
- customer profile updates;
- real-time scoring;
- PostgreSQL or MongoDB persistence;
- API for reading profiles and recommendations.

## Week 8: Portfolio Polish

Deliverables:

- bilingual README sections;
- architecture diagrams;
- screenshots or short demo videos;
- skill mapping table;
- interview pitch;
- limitations and next steps for each module.

## Week 9 Buffer

Use the buffer for:

- Databricks notebooks;
- cloud deployment on Azure or GCP;
- stronger Kubernetes/OpenShift manifests;
- monitoring and model drift examples;
- presentation rehearsal.
